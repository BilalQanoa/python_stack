# Flask HTML Table - Data Rendering 📊

A fundamental project focusing on how to pass structured data (Lists of Dictionaries) from a Flask backend to a frontend template and render it dynamically using Jinja2 loops.

---

### 💡 Features
* **Dynamic Data Rendering:** Iterating through a list of user dictionaries to generate table rows.
* **Full Name Concatenation:** Combining first and last names directly within the template.
* **Ninja Bonus (Bootstrap):** Styled the table using **Bootstrap 5** for a professional, responsive look.

---

### 🛠️ Key Concepts Learned
* **Backend-to-Frontend Data Flow:** Understanding how data travels from Python to HTML.
* **Jinja2 Loops:** Mastering the `{% for ... %}` syntax to handle multiple records.
* **CSS Frameworks:** Implementing Bootstrap classes like `table-striped` and `table-hover` to enhance UI with minimal effort.
